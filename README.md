## Install Bods

```sh
go install github.com/rollwagen/bods@latest
```
### Enable CLI complestion

ZSH example:

```sh
__BODS_CMP_ENABLED=1 bods completion zsh > b_cmp.sh; source b_cmp.sh; rm b_cmp.sh
```
